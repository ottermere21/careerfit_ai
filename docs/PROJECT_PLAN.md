## 1. 문제 
Q1. 취업 공고나 공모전 목록을 찾아보면서 있었던 문제점은?
- 여러 사이트에 공고가 흩어져 있다
- 회사 공고 밑에 회사 평점,후기랑 퇴사율 같은 지표가 보이면 좋겠다
- 나와 얼마나 관련있는지도 알려주면 좋겠다

Q2. CareerFit AI가 이 문제를 어떻게 해결하는가?
- 여러 사이트에 흩어져 있는 공고를 한 번에 정렬해서 보여주기
- 태그를 활용해서 요약을 해주기
- 내가 찾는 공고와 얼마나 맞는지 매칭률 표시하기


Q3. 이 서비스를 쓸 사용자가 누구인가?
